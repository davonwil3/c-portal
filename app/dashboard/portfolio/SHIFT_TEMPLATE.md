# Shift Editorial Template

## Overview
The **Shift** template is a bold, editorial-style portfolio design inspired by modern magazine layouts and creative agency websites. It features striking typography, strong visual hierarchy, and a contemporary aesthetic perfect for graphic designers, art directors, and creative professionals.

## Design Philosophy
Shift embraces **bold typography** and **editorial layouts** to create a portfolio that demands attention. The design draws inspiration from fashion magazines, creative studios, and contemporary design trends, prioritizing visual impact while maintaining excellent readability and user experience.

## Key Features

### Visual Elements
- **Massive Typography**: Extremely large headings (up to text-9xl) that create immediate visual impact
- **Editorial Layout**: Magazine-inspired layouts with asymmetric grids and dynamic compositions
- **Bold Borders**: Heavy (4px) borders used as design elements throughout
- **Beige/Gray Background**: Warm neutral background (#d4cfc4) that provides sophistication
- **Strong Hierarchy**: Clear visual hierarchy using size, weight, and spacing
- **Uppercase Headers**: All major headings in uppercase for bold, confident presentation
- **Minimal Color Palette**: Focus on typography and layout with accent colors used sparingly

### Layout Features
- **Info Bar Navigation**: Top horizontal bar with name, current role, location, and navigation links
- **Split Hero Layout**: Large typography on left, name display on right with supporting elements
- **Alternating Project Grids**: Projects alternate between left/right image placement
- **Numbered Services**: Services feature large number badges for visual interest
- **Border-Based Cards**: Heavy borders instead of shadows for service and content blocks
- **Hover Transformations**: Subtle translate-x animations on interactive elements

### Typography Approach
- **Display Text**: 7xl to 9xl for hero headings
- **Section Headers**: 5xl to 7xl for major sections
- **Body Copy**: Larger than average (xl) for excellent readability
- **Tracking**: Tight letter-spacing on display text, wide tracking on tags
- **Font Weight**: Heavy use of font-black (900 weight) for impact

## Customization Tips

### For Different Industries
1. **Fashion/Luxury**: Keep default beige background, use serif fonts
2. **Tech/Digital**: Use white background, increase spacing
3. **Art/Fine Arts**: Experiment with more vibrant primary colors
4. **Architecture**: Keep minimal, increase image sizes
5. **Editorial/Publishing**: Use Playfair or Lora fonts for authenticity

### Color Combinations
- **Classic**: Black text on beige (#d4cfc4) - default
- **High Contrast**: Pure black text on white
- **Warm**: Dark brown (#3e2723) on cream (#faf8f3)
- **Cool**: Dark navy (#1a237e) on light blue-gray (#eceff1)
- **Bold**: Use a vibrant primary color (red, blue, green) for accents

## Technical Notes

### Font Family Support
All standard fonts are supported. Recommended fonts for Shift:
- **System** (default): Clean, reliable
- **Inter**: Modern, geometric
- **Work Sans**: Bold, confident
- **Montserrat**: Urban, contemporary
- **Playfair Display**: Editorial, elegant (serif option)

### Image Recommendations
- **Hero Avatar**: Portrait orientation (4:5), 800x1000px minimum
- **Project Images**: Landscape (4:3), 800x600px minimum
- **High Quality**: Use sharp, professional images
- **Consistency**: Keep similar photography style across projects

### Performance
- Lightweight: No heavy graphics or animations
- Fast Loading: Optimized image recommendations
- Smooth Interactions: Minimal JavaScript, CSS-based animations
