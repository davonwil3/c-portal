# Shift Template - Color Strategy

## Overview
The Shift template now has strategic use of **Primary Color**, **Secondary Color**, **Text Color**, and **Background Color** to create a bold, editorial aesthetic with proper visual hierarchy.

## Color Applications

### Primary Color (#1a1a1a by default)
**Purpose**: Main brand color for key headings and CTA elements

**Used For**:
- Service titles (large, uppercase headings)
- Project titles (large, uppercase headings)  
- Contact icon colors
- CTA button background
- Main focal points that need emphasis

**Strategy**: Primary color draws attention to the most important content elements - the services you offer and projects you've completed.

---

### Secondary Color (#8b5cf6 - purple by default)
**Purpose**: Accent color for interactive elements, tags, and visual interest

**Used For**:
- **Arrow icon** in hero section (creates visual interest and guides eye flow)
- **Service number badges** (01, 02, 03) - solid background with white text
- **About section skill tags** - borders
- **Service tags** - borders
- **Project tags** - borders  
- **Testimonial left border** - accent stripe

**Strategy**: Secondary color creates rhythm and consistency across the page through repeated use in tags, badges, and accents. It provides visual variety without overwhelming the design.

---

### Text Color (#1a1a1a by default)
**Purpose**: Primary text and content color

**Used For**:
- All body copy
- Section headings (ABOUT, SERVICES, WORK, etc.)
- Hero tagline
- Large name display
- Navigation links
- Contact information
- Footer text
- Service descriptions
- Project summaries
- Testimonial quotes
- All readable content

**Strategy**: Text color maintains readability and consistency. It's slightly less intense than pure black for better eye comfort.

---

### Background Color (#d4cfc4 - warm beige/gray by default)
**Purpose**: Page background that provides warmth and sophistication

**Features**:
- **Fully customizable** via the appearance settings
- **Single background** throughout the entire page for minimalist cohesion
- **Neutral warm tone** that complements both primary and secondary colors
- **Contrasts well** with dark text for readability

**Strategy**: The warm, neutral background creates an editorial, sophisticated feel reminiscent of design magazines. It's less stark than white while maintaining excellent readability.

---

## Visual Hierarchy

### Level 1 (Highest Emphasis)
- **Service Titles** → Primary Color
- **Project Titles** → Primary Color
- **CTA Button** → Primary Color background

### Level 2 (Secondary Emphasis)
- **Service Number Badges** → Secondary Color background (bold, eye-catching)
- **Arrow Icon** → Secondary Color (guides visual flow)
- **All Tags/Badges** → Secondary Color borders

### Level 3 (Content)
- **All Body Text** → Text Color
- **Section Headers** → Text Color (large, uppercase, bold)
- **Descriptions & Details** → Text Color

---

## Design Benefits

### 1. **Clear Visual Hierarchy**
Primary color highlights what you do (services) and what you've done (projects). Secondary color adds rhythm and modernizes traditional tags.

### 2. **Bold Editorial Feel**
The contrast between dark primary colors and vibrant secondary accents creates the bold, confident aesthetic expected in editorial design.

### 3. **Consistency Through Repetition**
Secondary color appears consistently in tags across all sections (about, services, projects), creating visual unity.

### 4. **Flexibility**
All colors are customizable via the appearance panel, allowing users to:
- Match their brand colors
- Create light or dark themes
- Adjust for different industries
- Experiment with color psychology

## Customization Tips

### For Tech/Startup
- Primary: #000000 (Pure Black)
- Secondary: #3b82f6 (Blue)
- Background: #f5f5f5 (Light Gray)

### For Creative/Agency
- Primary: #1a1a1a (Default)
- Secondary: #8b5cf6 (Purple - Default)
- Background: #d4cfc4 (Warm Beige - Default)

### For Luxury/Fashion
- Primary: #000000 (Black)
- Secondary: #d4af37 (Gold)
- Background: #faf8f3 (Cream)

### For Health/Wellness
- Primary: #1a5f3f (Forest Green)
- Secondary: #7dd3c0 (Mint)
- Background: #f0f4f2 (Pale Green)

## Images in Shift Template

### Hero Section
- **Avatar Image**: 4:5 aspect ratio portrait
- Location: Left side below tagline
- Editable via InlineImageReplace component

### Projects Section
- **Project Cover Images**: 4:3 aspect ratio
- Layout: Alternating left/right with text
- Hover effect: Subtle scale-up transition
- Editable via InlineImageReplace component

All images are fully functional with inline editing capabilities in edit mode.
