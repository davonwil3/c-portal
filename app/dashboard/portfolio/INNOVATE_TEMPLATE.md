# Innovate Template

## Overview

The **Innovate Template** is a modern, corporate-focused portfolio template designed for tech companies, digital agencies, and innovative businesses. It features a bold hero section with stats overlay, clean card-based layouts, and a professional aesthetic that communicates trust and innovation.

## Design Inspiration

The template is inspired by modern corporate websites and SaaS landing pages, featuring:
- Large, impactful hero section with overlay stats
- Clean white background with accent colors
- Card-based component layouts with hover effects
- Rounded corners and modern shadows
- Professional typography and spacing
- Stats showcase (e.g., "10+ Years of Experience", "70+ Successful Projects")

## Key Features

### Hero Section
- **Large Hero Image**: Full-width hero image with gradient overlay
- **Stats Overlay**: Display key metrics directly on the hero image (customizable)
- **Top Navigation**: Logo on left, "Get In Touch" CTA button on right
- **Bold Headline**: Large, attention-grabbing headline split across two lines
- **Prefix Badge**: Optional pill-shaped badge above the headline

### Services Section
- **Card Grid Layout**: 3-column grid on desktop, responsive on mobile
- **Icon Placeholders**: Color-matched icon boxes for each service
- **Hover Effects**: Cards lift with shadow on hover
- **Clean Typography**: Bold headings with descriptive text

### Projects Section
- **2-Column Grid**: Large project cards with cover images
- **Image Hover**: Images scale up on hover for dynamic effect
- **Tag Pills**: Colored pill-shaped tags for project categories
- **Gray Background**: Section uses light gray background for contrast

### Testimonials Section
- **3-Column Grid**: Compact testimonial cards
- **Quote Icons**: Visual quote marks in brand color
- **Avatar Circles**: Circular avatar placeholders with initials
- **Author Details**: Name and role clearly displayed

### Contact Section
- **Split Layout**: Contact info cards on left, form on right
- **Icon Cards**: Contact items displayed in elevated cards with icons
- **Modern Form**: Large, rounded inputs with good spacing
- **Gray Background**: Matches projects section for visual rhythm

## Color Scheme

## Typography

- **Font Family**: System font stack (customizable)
- **Heading Sizes**: 
  - Hero: 5xl to 7xl (responsive)
  - Section Headings: 4xl to 5xl
  - Card Titles: 2xl to 3xl
- **Font Weights**: Bold for headings, regular for body text

## Layout Structure

1. **Hero Section** (Full viewport height)
2. **Services Section** (White background)
3. **Projects Section** (Gray background)
4. **Testimonials Section** (White background)
5. **Contact Section** (Gray background)
6. **Footer** (White background with border)

## Customization

All elements are fully customizable through the portfolio builder:
- Edit text inline (click to edit)
- Replace images by clicking on them
- Change colors through the appearance panel
- Reorder sections via the control panel
- Add/remove services, projects, and testimonials through modals
- Change icons for contact items

## Stats Configuration

The hero section stats are configured using the `about` object:
- `about.column1`: First stat number (e.g., "10+")
- `about.column2`: First stat label (e.g., "Years of Experience")
- `about.heading`: Second stat number (e.g., "70+")
- `hero.bio`: Second stat label (e.g., "Successful Projects")

## Responsive Design

The template is fully responsive:
- **Desktop (1400px+)**: Full multi-column layouts
- **Tablet (768px-1399px)**: 2-column grids, adjusted spacing
- **Mobile (<768px)**: Single column, stacked layout, touch-friendly

## Technical Details

- Built with React and TypeScript
- Uses Tailwind CSS for styling
- Fully accessible with semantic HTML

- SEO-friendly structure
- Edit mode with inline editing capabilities

## Default Content

The template comes pre-populated with:
- 3 Services (Digital Transformation, Custom Software, AI & Analytics)
- 4 Projects (Cloud Migration, AI Platform, E-Commerce, Healthcare)
- 3 Testimonials from various roles
- 4 Contact items (Email, Phone, Location, LinkedIn)

All default content can be modified or replaced through the portfolio builder interface.

## Creating an Innovate Portfolio

1. Navigate to Portfolio Builder
2. Click "Change Template"
3. Select "Innovate Corporate"
4. Customize your content:
   - Update hero headline and stats
   - Add your hero image
   - Edit services, projects, and testimonials
   - Customize colors and fonts
5. Preview and publish
