# Minimalist Template - Strategic Color Usage

## Design Philosophy

The Minimalist template now follows a **strategic color hierarchy** to maintain true minimalism:
- **Black (#000)** - Strong hierarchy elements
- **Primary Color** - Sparingly used for key actions and highlights
- **Text Color** - Majority of content for cohesion
- **Secondary Color** - Available but not actively used (minimal aesthetic)

## Customization Strategy

### For Light Backgrounds
**Recommended:**
- Text Color: Dark gray (#1f2937 default) or darker
- Primary Color: Any brand color
- Creates clear contrast

### For Dark Backgrounds
**Recommended:**
- Text Color: Light gray (#e5e7eb) or white
- Primary Color: Lighter brand color for visibility
- Background Color: Dark tone

### For Colored Backgrounds
**Recommended:**
- Text Color: Complementary color with good contrast
- Primary Color: Accent that stands out
- Test readability

## Summary

The Minimalist template now uses a **strategic three-tier color system**:

1. **Black** - Strong hierarchy (headings, titles, key info)
2. **Primary Color** - Strategic highlights (CTAs, pricing, accents)
3. **Text Color** - Foundation (majority of content)

This creates a truly minimal, professional, and highly customizable portfolio that maintains visual clarity while giving users complete control over their brand presentation.
